# Gon-dotfiles

## Requisitos

### Arch / CachyOS

```bash
sudo pacman -Syu
sudo pacman -S --needed git
```

### Instalación

```bash
git clone https://github.com/GonSmrt/Gon-dotfiles.git
cd Gon-dotfiles
./install.sh
```

####Este script instala y configura automáticamente:

- Zsh
- Oh My Zsh
- Powerlevel10k
- Fastfetch
- Kitty
- Wallpapers
- Fuentes
- Temas e iconos
- Dotfiles personales
